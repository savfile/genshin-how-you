# 原神怎么你了？

一个哔哩哔哩弹幕网的弹幕屏蔽词配置文件，用于屏蔽 _~~原批~~_ 有关原神的弹幕。

本配置文件可能会 _**严重误伤**_ 无关弹幕，请谨慎使用。
